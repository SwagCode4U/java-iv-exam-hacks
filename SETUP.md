## 💻 Prerequisites

### What You Need

| Tool | Version | Purpose |
|------|---------|---------|
| Java Development Kit (JDK) | 8 or higher | Write and run Java code |
| Text Editor / IDE | - | VS Code, IntelliJ, Eclipse, or Notepad |
| Git | Latest | Version control (optional) |

### Installing Java

#### Windows
1. Go to [oracle.com/java](https://www.oracle.com/java/technologies/downloads/)
2. Download JDK for Windows
3. Run the installer
4. Set `JAVA_HOME` environment variable

#### Linux (Ubuntu/Debian)
```bash
sudo apt update
sudo apt install openjdk-11-jdk
java -version
```

#### macOS
```bash
# Using Homebrew
brew install openjdk@11
java -version
```

### Verify Installation
```bash
java -version
javac -version
```

## 🖥️ Running Java Code

### Method 1: Command Line

```bash
# Create a file
nano HelloWorld.java

# Write your code
# class HelloWorld {
#     public static void main(String args[]) {
#         System.out.println("Hello, World!");
#     }
# }

# Compile
javac HelloWorld.java

# Run
java HelloWorld
```

### Method 2: Using IDE

#### VS Code
1. Install "Java Extension Pack"
2. Create new file with `.java` extension
3. Right-click → Run Java

#### IntelliJ IDEA
1. File → New → Project
2. Select Java
3. Create class with main method
4. Click Run

### Method 3: Online Compilers
- [JDoodle](https://www.jdoodle.com/online-java-compiler/)
- [Programiz](https://www.programiz.com/java-programming/online-compiler/)
- [Replit](https://replit.com/languages/java10)

## 📝 Exam Tips

✅ Eat a healthy breakfast
